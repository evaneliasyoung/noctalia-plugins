# Noctalia Plugins

## Add This Repository As a Source

1. Open Noctalia.
2. Go to the plugins section.
3. Add a custom plugin source.
4. Use the repository URL of this source repo:

```text
https://github.com/evaneliasyoung/noctalia-plugins
```

5. Install the plugin you want from the list.

## Plugins

<details>
<summary><strong>Nightscout Widget</strong> — CGM remote monitor</summary>

### What it does

Provides a `BarPill` with the latest reading from the Nightscout API.

It supports:

- different colors for each type of reading (in-range, out-of-range, urgent)
- showing the value on hover, always, or never

</details>
